# Dashboard Features Reference

A complete feature inventory for all three LifeRise portals.

---

## Vendor Portal — `/vendor`

### Dashboard Overview
- Personalized greeting with live date and time
- Real-time Online / Offline availability toggle with pulsing status indicator
- Summary KPI strip: active jobs today, weekly earnings, star rating, completion rate

### Booking Queue (`/vendor/queue`)
- Kanban-style board with three columns: **Incoming**, **In Progress**, **Completed**
- Accept or decline booking requests with animated card transitions
- Client name, service type, scheduled time, and address displayed per card
- Queue count badges per column updated in real-time

### Schedule (`/vendor/schedule`)
- Daily timeline view of all confirmed appointments
- Per-booking detail: client name, service, time slot, duration, and live status
- Active appointment highlighted with teal accent
- Upcoming appointment preview list

### Earnings (`/vendor/earnings`)
- Weekly bar chart showing revenue per day (gold accent for peak day)
- Monthly revenue total with percentage change vs. prior month
- Per-job earnings log with service name, client, date, and payout amount
- Payout status indicators: pending, processing, paid

### Services (`/vendor/services`)
- List of offered services with pricing, duration, and category
- Toggle individual services active or inactive
- Edit pricing and description per service

### Profile (`/vendor/profile`)
- Display name, avatar initials, specialty, and bio
- Star rating aggregate and total review count
- Service area / location field
- Account settings: notification preferences, payout details

### Promo Code System
- Share vendor-specific promo codes with clients to incentivize first bookings
- `VENDOR50` — 50% off first month of premium listing placement
- `REFER20` — $20 credit per new vendor referral

---

## Resident Portal — `/resident`

### Dashboard Overview
- Personalized greeting with live date
- KPI strip: active bookings, pending bookings, completed bookings, total spend
- Service category filter chips (Wellness, Maintenance, Lifestyle, Cleaning, Fitness…)
- Featured vendor cards with ratings, specialty, and one-tap Book Now action

### Services (`/resident/services`)
- Full service catalog browsable by category
- Vendor cards: name, initials, rating, specialty, price, and badge (e.g., Top Rated)
- Search and filter by service type, price range, and availability
- One-tap booking flow: select time slot → confirm → done

### Bookings (`/resident/bookings`)
- Active, pending, and completed booking tabs
- Per-booking detail: service name, vendor, scheduled date, status badge, price
- Status color coding: Confirmed (teal), Pending (gold), Completed (purple)
- Cancel or reschedule option on upcoming bookings

### Events (`/resident/events`)
- Community event feed: title, date, location, and description
- RSVP toggle per event
- Upcoming vs. past events separated by tab

### Favorites (`/resident/favorites`)
- Saved vendor cards for quick rebooking
- Remove from favorites with a single tap
- Most recently favorited vendors shown first

### Notifications (`/resident/notifications`)
- Booking confirmations, status updates, and vendor messages
- Community announcements from property management
- Read / unread state with timestamp

### Profile (`/resident/profile`)
- Display name, unit number, and contact details
- Saved payment methods list
- Booking history summary
- Notification and privacy preferences

### Promo Code System
- Enter promo codes at checkout for instant discounts
- `WELCOME25` — 25% off first booking
- `REFER15` — $15 credit when a referred neighbor completes their first booking

---

## Manager Portal — `/manager`

### Dashboard Overview
- Property name header with live date
- KPI strip: total residents, active vendors, monthly bookings, satisfaction score
- Trend indicators per KPI (e.g., +4 residents, +18% bookings)
- Engagement donut chart: service category breakdown as percentage of total bookings
- Top Vendor leaderboard: rank, name, rating, and earnings
- Latest community announcement preview

### Analytics (`/manager/analytics`)
- Booking volume trend line chart (weekly / monthly toggle)
- Revenue generated by service category (bar chart)
- Resident satisfaction score over time
- Vendor performance comparison table: jobs completed, rating, earnings, compliance status
- Exportable summary metrics

### Residents (`/manager/residents`)
- Searchable resident directory with unit number, move-in date, and booking activity
- Individual resident profile: contact info, active bookings, payment status
- Resident engagement score per unit

### Vendors (`/manager/vendors`)
- Vendor application queue: approve or reject new vendor submissions
- Active vendor list with compliance status (certified / lapsed / pending review)
- Vendor leaderboard: ranked by rating, bookings, and earnings
- Per-vendor detail: services offered, schedule availability, and review history

### Announcements (`/manager/announcements`)
- Compose and broadcast property-wide announcements
- Target audience selector: all residents, all vendors, or both
- Announcement history with sent date and read-receipt count
- Pin priority announcements to top of resident feed

### Property Map (`/manager/settings` — map component)
- Interactive Leaflet-based map of the property and surrounding area
- Vendor location pins with service category color coding
- Resident unit overview layer

### Brand Personalization
- Every manager dashboard is customized to the property's brand identity
- Property logo, accent color, and name displayed throughout the portal
- Bespoke feature development available for custom integrations and white-label portals

### Promo Code System
- Generate property-wide promo codes for resident onboarding campaigns
- Track redemption rates and campaign effectiveness per code
- `LAUNCH50` — property-wide launch promo distributed to all residents
- `MANAGER10` — 10 free resident invite codes for move-in onboarding batches
